Tutorial for run this project


1. Open terminal paster git clone https://github.com/Saifullakh1/cs_22
2. Open folder cs_22 creat venv python3 -m venv env
3. Activate venv source env/bin/activate
4. Install requirements pip3 install -r requirements.txt
5. Migrations python3 manage.py migrate
6. Create superuser python3 manage.py createsuperuser
7. Run project python3 manage.py runserver
